# 配分(色分け)データ更新

Excel の色分け表から `src/distribution.js` の `DISTRIBUTION_GROUPS` を更新するスキル。

## 前提知識

- 配分表示(モード2)は、ボランティアのチームごとに担当する掲示場所を色分けして表示する機能。
- `DISTRIBUTION_GROUPS` の各要素: `name`(色名)/ `fill` / `stroke` / `textColor` / `ids`(担当する掲示場所の id 配列)。
- ユーザーは Excel のセル背景色で配分を管理している。渡されるのは「色名 → id リスト」の形が多い。
- 全 240 件のうち配分されるのは一部だけでよい(2026-07 時点: 8色118件)。配分外の id はモード2で非表示になる。

## 色名 → 表示色の対応表(既存の運用)

新しい色が来たらこの表に従い、無い色は視認性(白地図上でのコントラスト、ラベル文字色)を考えて決め、この表に追記する。

| 色名 | fill | stroke | textColor |
|---|---|---|---|
| 白 | #ffffff | #999999 | #000000 |
| 黒 | #cccccc | #666666 | #000000 |
| 茶 | #85200c | #ffffff | #ffffff |
| みどり | #00ff00 | #ffffff | #000000 |
| ピンク | #ff00ff | #ffffff | #000000 |
| 青 | #3c78d8 | #ffffff | #ffffff |
| 赤 | #ff0000 | #ffffff | #ffffff |
| オレンジ | #ff9900 | #ffffff | #000000 |

※「黒」は視認性のため実際には灰色 (#cccccc) で表示する運用になっている点に注意。

## 手順

1. ユーザーから受け取った色分け(Excel 貼り付け・CSV・箇条書きいずれも可)を「色名 → id リスト」に整理する。
2. id の並び順は**受け取った順のまま**にする(Excel 上の並び=回る順の可能性があるため、勝手にソートしない)。
3. `src/distribution.js` を書き換える。
4. `node scripts/validate-data.mjs` を実行し、以下を確認する:
   - すべての id が data.js に存在すること
   - 同じ id が複数の色に入っていないこと
5. 色ごとの件数と合計をユーザーに報告する(Excel 側と件数を突き合わせてもらうため)。

## 必ず守ること

- 配分から外れた id は「削除された」のか「まだ未配分」なのか区別がつかないため、既存グループから id が消える変更のときは差分を明示して報告する。
- 色の追加・変更をしたら、この SKILL.md の対応表も更新する。
